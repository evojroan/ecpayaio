# 這是什麼

這是簡單的前端示範網頁，示範如何以前端購物網頁串接綠界科技的全方為支付金流(AIO)

# 如何操作本網頁

請輸入購買數量、收件者姓名、收件者地址，按下「按我結帳」就能連結至綠界支付頁面。

# 支付頁面僅為模擬頁面

非實際付費頁面。請使用以下信用卡號碼結帳：
4311-9522-222-222
安全碼：222
日期：晚於現今年月份即可。
